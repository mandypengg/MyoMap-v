# MyoMap

An interactive 3D human anatomy web app that lets you explore and learn about major muscle groups. Click on any muscle on the 3D model to navigate to a dedicated page with information on muscle function and recommended exercises.
<br><br>
🔗 Try it here: [https://myomap.vercel.app/](url)

## Features

- **Interactive 3D Model**  
   Rotate a full human anatomy model and click on muscle hotspots to learn more.

- **Muscle Pages**  
   Each major muscle group has its own page covering muscle function and targeted exercises.

- **Sidebar**  
  Slide-out menu with sources, and contact/social links.
  
- **Sources Panel**
  All referenced academic and clinical sources linked directly in the app
  
- **Simple and Intuitive UI**  
  Designed with usability in mind, MyoMap features a straightforward interface that allows users to navigate the model and website easily.

## Motivation

Weightlifting is my passion. I constantly see misinformation about muscles online especially on social media, so I designed MyoMap to help streamline beginner's fitness journeys.
